this class learn. create GitHub repository and connect local file repository in my computer.



GitHub repository connect local file step by step

  1. open terminal
  2. and open file directory
  3. than type | git init
  4. than create a file for this folder
  5. than type | git add .
  6. than type | git commit -m "meaning message"
  7. than type | git branch -m main
  8. than type | git remote origin (github origin link )
  9. and last of type | git push -u origin main
  10. finally connect local folder to GitHub
